# 🎯 Domain Recommendation System

An ML-based web app that recommends the best domain (career path) based on user skills.

---

## 🚀 Overview

This project uses a **Random Forest Classifier** to analyze user skills and predict the most suitable domain.
The app is built with **Streamlit**, making it interactive and easy to use.

---

## 🧠 Features

* Machine Learning model (Random Forest)
* Interactive UI using Streamlit
* Real-time domain prediction
* Simple and user-friendly interface

---

## 📊 Input Features

Users provide skill ratings (1–10) for:

* Coding
* Math
* Creativity
* Communication

---

## 🎯 Output

* Recommended domain based on input skills

---

## 🛠️ Tech Stack

* Scikit-learn

## ⚙️ Installation & Run Locally

1. Clone the repository:
   git clone https://github.com/your-username/domain-recommendation-system.git

2. Navigate to the folder:
   cd domain-recommendation-system

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app.py

## 📈 Model Details

* Algorithm: Random Forest Classifier
* Train-Test Split: 80-20
* Evaluation Metric: Accuracy Score
